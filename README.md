# Mac のキッティング

最初に homebrew を設定します。
https://brew.sh

git コマンドがインストールされたので、このレポをクローンします (ssh -A で主マシンからログインして実行すると便利です)
環境設定 - 共有 から Remote login 、画面共有を有効にしておくと便利です
`git clone git@github.com:taiyodayo/mysettings.git`

セットアップを実行します
```
cd ~/mysettings
./setup_mac_all.sh
```
